Zero-Based Normalization (0 to Max)

Advantages:
  • Shows the true relative scale of each cryptocurrency compared to its maximum value
  • Makes it easier to see the percentage of maximum value each cryptocurrency is at
  • Provides a consistent baseline (0) for all cryptocurrencies
  • Better represents the actual price movement relative to the maximum price
Disadvantages:
  • If a cryptocurrency's minimum price is far from zero, a large portion of the chart space might be unused
  • Small fluctuations might appear less significant visually

Min-Max Normalization (Min to Max)

Advantages:
  • Maximizes the use of the vertical chart space
  • Makes small fluctuations more visible
  • Better for comparing volatility patterns between cryptocurrencies
Disadvantages:
  • Distorts the true relative scale between cryptocurrencies
  • A small absolute change in a stable cryptocurrency can appear as dramatic as a large change in a volatile one
  • Harder to compare the actual percentage changes between cryptocurrencies
